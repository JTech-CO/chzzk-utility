Chzzk Utility 개인정보 처리방침

최종 업데이트: 2026년 5월 12일

1. 개인정보 수집 및 이용

Chzzk Utility는 사용자 개인을 식별할 수 있는 어떠한 개인정보도 수집, 저장, 전송하지 않습니다.

본 확장 프로그램은 오직 네이버 치지직(Chzzk) 라이브 및 다시보기 시청 환경을 개선하기 위한 유틸리티 목적으로만 동작합니다.

2. 권한(Permissions) 사용 목적

확장 프로그램이 원활히 동작하기 위해 구글 크롬 웹스토어 정책에 따라 다음과 같은 최소한의 권한을 요청합니다:

  • declarativeNetRequest: 광고 관련 네트워크 요청(veta.naver.com 등)을 정적 룰셋으로 차단하기 위해 사용됩니다. 사용자의 브라우징 데이터를 수집하거나 외부로 전송하지 않습니다.
  • storage: 사용자가 팝업에서 설정한 옵션(광고 차단, 화질 고정, 타임머신 등)을 브라우저 로컬 저장소에 저장하기 위해 사용됩니다. 저장된 데이터는 사용자의 디바이스에만 보관됩니다.
  • scripting: 치지직 페이지에 유틸리티 스크립트를 주입하여 API 응답 변조(광고 제거, 타임머신 활성화) 및 UI 컨트롤을 삽입하기 위해 사용됩니다.
  • 호스트 권한 (*.naver.com 등): 치지직 API 응답을 가져와서 광고 메타데이터를 제거하고, 타임머신 기능을 강제 활성화하기 위해 필요합니다. 이러한 처리는 사용자의 브라우저 내에서만 이루어지며 외부 서버로 어떠한 데이터도 전송되지 않습니다.

3. 데이터 처리 방식

본 확장 프로그램은 치지직 API 응답을 브라우저 내에서 실시간으로 가공하여 광고를 제거하고 기능을 활성화합니다. 이 과정에서:

  • 사용자의 로그인 정보, 시청 기록, 개인 식별 정보에 일절 접근하지 않습니다.
  • 네트워크 요청 차단은 선언적(declarative) 방식으로만 이루어지며, 요청 내용을 읽거나 기록하지 않습니다.
  • 모든 데이터 처리는 사용자의 로컬 브라우저 내에서 완료되며, 제3자 서버를 경유하지 않습니다.

4. 개인정보 처리방침의 변경

본 확장 프로그램의 업데이트나 크롬 웹스토어 정책 변경 등에 따라 이 개인정보 처리방침의 내용이 변경될 수 있습니다. 변경사항이 발생할 경우 버전 업데이트를 통해 안내되며, 본 페이지가 지속적으로 최신화될 것입니다.
